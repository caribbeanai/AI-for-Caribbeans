# caribbean-localise-recipe

Adapt any recipe to ingredients and units common in a target Caribbean country.

## When to use

- The user pastes a recipe and asks for help making it locally.
- The user wants substitutions for hard to find ingredients.

## Steps

1. Identify the dish and core technique.
2. Replace hard to source ingredients with closest local equivalents.
3. Convert units to local conventions (Celsius oven, lbs and oz often, sometimes grams).
4. Adjust quantities for the requested number of people.
5. Suggest one local side that pairs well.
6. Estimate total cost in local currency where possible.

## Inputs

- Recipe text
- Country
- Number of people

## Output format

Markdown sections: Ingredients (with swaps), Method, Notes on flavour, Cost estimate.
