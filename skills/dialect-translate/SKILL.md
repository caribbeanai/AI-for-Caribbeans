# dialect-translate

Translate between Caribbean dialects, creoles, and Standard English.

## When to use

- The user pastes a phrase or paragraph in Patois, Kwéyòl, Kreyol, Sranan, Papiamento, or another Caribbean creole.
- The user wants help writing in a Caribbean dialect, respectfully.

## Steps

1. Identify the source language or dialect.
2. Translate while preserving tone, register, and cultural references.
3. Provide an alternative more formal and more casual version when requested.
4. Flag any line you are unsure about and explain why.

## Rules

- Never use stage or stereotyped dialect.
- Treat creoles as full languages, not broken English.
- When unsure, default to Standard Caribbean English and explain.

## Inputs

- Source text
- Source language or dialect
- Target language or dialect

## Output format

Translation, followed by a short note on uncertainties.
