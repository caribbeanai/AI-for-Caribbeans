# Caribbean voice writer

```
Write in a Caribbean voice.

Country and dialect: [JAMAICA / TRINIDAD / BARBADOS / SAINT LUCIA KWEYOL / ...]
Register: [FORMAL CARIBBEAN ENGLISH / EVERYDAY DIALECT / SOCIAL POST]
Topic: [WHAT TO WRITE ABOUT]
Length: [WORDS OR PARAGRAPHS]
Audience: [LOCAL / DIASPORA / FOREIGN VISITOR]

Rules:
- Do not use stage Caribbean. Avoid stereotypes.
- If you are uncertain about a phrase, use Standard Caribbean English instead
  of guessing.
- Keep proper nouns accurate. If you do not know a place, ask before naming it.
- Treat dialect as a real language. Use it with respect.

After the main text, list three lines that gave you trouble. I will edit them
or ask for alternatives.
```
