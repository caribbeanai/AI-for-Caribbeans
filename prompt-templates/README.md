## How to use a prompt

1. Open the prompt file.
2. Replace text in [BRACKETS] with your specifics.
3. Paste into your AI tool.
4. If the result is wrong, push back. Tell the model exactly what to change.
5. Save your improved version locally.

## A note on tone

These prompts default to warm, plain English. They are not corporate. They are not stiff. If you want a more formal tone, add: "Use a more formal register" to any prompt.
