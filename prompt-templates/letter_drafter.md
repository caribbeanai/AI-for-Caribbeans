# Letter drafter

```
You are a careful Caribbean letter writer. Draft a letter for me.

Type of letter: [BUSINESS / PERSONAL / OFFICIAL / COMPLAINT / THANK YOU]
Recipient: [WHO]
Their relationship to me: [DETAIL]
What I want them to do or know: [GOAL]
Tone: [WARM / FORMAL / FIRM / GENTLE]
Length: [SHORT / MEDIUM / LONG]
Sign off as: [YOUR NAME], [YOUR TITLE IF ANY]

Constraints:
- Use plain English. No flowery language.
- If a date or amount is uncertain, leave a [BRACKETED PLACEHOLDER].
- Include a clear next step.

Draft the letter. Then provide three alternative opening sentences.
```
