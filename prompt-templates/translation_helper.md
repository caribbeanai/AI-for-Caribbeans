# Translation helper

```
Translate the following text between [SOURCE LANGUAGE] and [TARGET LANGUAGE].

Text:
[PASTE]

Rules:
- Keep tone and register.
- Preserve cultural references. If untranslatable, use a footnote.
- For Caribbean creoles (Patois, Kweyol, Kreyol, Sranan, Papiamento),
  prefer everyday spoken style over academic forms.
- If the source mixes languages (code switching), preserve the mix unless
  I ask you to standardise.

Output:
- Translation
- A short note on any meaningful loss in translation
- If asked, provide a more formal and a more casual version
```
