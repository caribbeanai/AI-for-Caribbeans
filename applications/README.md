# Applications

Each subfolder solves a real Caribbean problem. Open a folder, read its README, run the code.

- [climate-change](climate-change) Hurricanes, sea level, coral.
- [personal-finance](personal-finance) Budgets, remittances, savings.
- [agriculture](agriculture) Yields, pests, prices.
- [tourism](tourism) Demand, sentiment, pricing.
- [health](health) Diabetes, dengue, mental health.
- [disaster-preparedness](disaster-preparedness) Routing, damage assessment.
- [education](education) Tutoring, lesson plans, study aids.
- [entrepreneurship](entrepreneurship) Pricing, content, support bots.
