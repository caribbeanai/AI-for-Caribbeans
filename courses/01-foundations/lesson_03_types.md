# Lesson 3, Types of AI

Three lenses to organise the field.

## By learning style

- **Supervised learning.** We show the model many input-output pairs. It learns the mapping. Example: photos of mangoes labelled Julie or East Indian, then predict the type of a new photo.
- **Unsupervised learning.** We show inputs only. The model finds structure. Example: group ten thousand tourist reviews by theme without telling it what the themes are.
- **Reinforcement learning.** The model acts, gets a reward, and learns a policy. Example: a delivery routing system that learns the fastest path through Port of Spain traffic.

## By data type

- Tabular. Spreadsheets, financial data, weather observations.
- Text. Reviews, emails, news articles, social posts.
- Image. Reef photos, drone surveys, identity documents.
- Audio. Calls, dialect recordings, bird songs.
- Video. CCTV, sports footage, drone footage.
- Multimodal. Combinations of the above. Modern LLMs handle this.

## By capability

- Narrow AI. Does one thing well. The mango sorter is narrow.
- General purpose models. Do many tasks, like Claude or GPT.
- Agents. Use tools, take multi step actions, write and run code, plan and act.

## Match the task to the type

| Task | Best fit |
|------|---------|
| Forecast cruise arrivals for Saint Maarten | Supervised ML, tabular |
| Detect fraud in remittance transactions | Supervised ML, tabular |
| Group customer reviews into themes | Unsupervised ML, text |
| Identify a banana plant disease from a photo | Deep learning, image |
| Draft a tour itinerary | Generative AI, text |
| Translate a Kreyol letter | Generative AI, text |
| Optimise hotel room pricing each day | Reinforcement learning or supervised plus rules |
