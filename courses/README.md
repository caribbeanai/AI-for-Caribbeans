# Courses

Six progressive courses. Each course has a README, lessons, code, and a Caribbean capstone.

| Course | Level | Capstone |
|--------|-------|----------|
| [01 Foundations](01-foundations) | Beginner | Build an AI literacy poster for your school |
| [02 Supervised ML](02-supervised-ml) | Beginner to Intermediate | Predict rainfall in Saint Lucia |
| [03 Unsupervised ML](03-unsupervised-ml) | Intermediate | Cluster Caribbean tourism reviews |
| [04 Deep Learning](04-deep-learning) | Intermediate | Classify reef health from photos |
| [05 LLMs](05-llms) | Intermediate to Advanced | Build a patois aware chatbot |
| [06 GenAI](06-genai) | Advanced | Ship a small AI product |

Work through them in order, or jump straight to the topic you need. Every course has a `scenarios.md` file with Caribbean stories mapped to datasets in [../datasets/CATALOG.md](../datasets/CATALOG.md).
