# Caribbean image prompts playbook

Twelve prompts that produce strong Caribbean imagery. Drop into any image model.

## Tourism

```
A small wooden fishing boat on a calm turquoise bay at dawn, Soufriere, Saint Lucia,
the Pitons in the background, fishermen mending nets, sun bleached colours,
shot on 35mm film, natural light, wide aspect ratio.
```

## Food

```
Top down photo of a wooden table with bake and shark, doubles, callaloo, plantain,
fresh coconut water, on a banana leaf, midday natural light, vibrant, editorial style.
```

## Music

```
Steel pan orchestra in performance, Queen's Park Savannah at dusk, Port of Spain,
players in colourful uniforms, motion blur on hands, warm stage lights, photojournalism style.
```

## Festival

```
Junkanoo dancer mid leap on Bay Street, Nassau, brilliant feathered costume,
crowd in soft focus behind, golden hour, editorial fashion lighting.
```

## Climate

```
Aerial view of a coral reef off Speightstown, Barbados, half the frame bleached,
half still vivid, sun rays through clear water, scientific photography style.
```

## Agriculture

```
A young farmer with a tablet inspecting banana plants in a rural valley, Saint Vincent,
morning mist, sustainable agriculture mood, documentary style.
```

## Education

```
Schoolchildren in uniform laughing in a sunny classroom, Bridgetown, Barbados,
chalkboard with maths problems, warm natural light, photojournalism.
```

## Heritage

```
Maroon village in the hills of Cockpit Country, Jamaica, traditional structures,
elder telling a story to children, soft afternoon light, oil painting style.
```

## Finance

```
A small fintech founder at a co-working desk, Kingston, Jamaica,
two monitors with charts, plants in the background, modern minimal style.
```

## Wildlife

```
A Sisserou Parrot perched on a branch in dense Dominica rainforest,
sunlight filtering through canopy, sharp focus on bird, nature documentary style.
```

## Architecture

```
Pastel coloured Dutch colonial buildings along Punda waterfront, Willemstad, Curaçao,
late afternoon, gentle reflections in the harbour, architectural photography.
```

## People

```
Three generations of a Caribbean family on a verandah, grandmother, mother, daughter,
laughing together, golden hour, soft skin tones, photographic, editorial portrait style.
```
