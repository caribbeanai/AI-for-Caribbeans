# Lesson 5, Code generation as a daily tool

If you write code, AI coding tools change the shape of your day.

## Three modes

1. **Autocomplete.** Inline suggestions while you type. Cursor, GitHub Copilot, JetBrains AI.
2. **Conversational coding.** Ask an LLM in chat, paste code, iterate.
3. **Agentic coding.** A coding agent reads the repo, plans, edits files, runs tests. Claude Code, Aider, Cline.

## How to be good at this

- Read the code the AI produces. Do not commit what you do not understand.
- Write tests. Then ask the AI to make them pass.
- Keep one source of truth. The AI follows your conventions when your codebase is clean and consistent.
- Use commit messages to record human intent.

## Caribbean caution

If you are remote and billing in USD, leverage these tools. They multiply you. But never paste client confidential code into a personal account. Use enterprise accounts or self host.
